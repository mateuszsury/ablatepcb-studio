AblatePCB Studio 0.2.2 improves the Gerber control preview.

Highlights:

- the complete PCB outline is always fitted inside the preview frame;
- the frame follows the actual board aspect ratio;
- TOP, BOTTOM, and overlay previews use consistent protected margins;
- long source paths no longer widen the workspace on narrow screens;
- desktop and mobile layouts were validated with a real 62 x 30 mm Gerber project.

This update changes preview presentation only. Generated mask geometry and laser controls are unchanged.
